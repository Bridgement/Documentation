Embedded Sign-up for Bridgement Partners

Overview

Bridgement has the ability for partners to use our sign-up process embedded in their own website. All that is required is for the partner to use a javascript file created by Bridgement and input a reference that we provide to you as part of the setup.

Basic Installation

Copy and paste the following snippet on every page you want to allow the embedded sign-up on.

<script src="https://integration.bridgement.com/embedded-sign-up/v1/index.js"></script>
<script>
    embeddedSignUp.settings = { clientReference: "supplied-company-reference" };
</script>

Initiating a sign-up

Use the following snippet to initiate the sign-up process. This should create an inline iFrame in the partner’s window, where the user will be able to sign-up to Bridgement from the partner’s website.

embeddedSignUp.signup();

Domain Registration

In order to enable the embedded sign-up on the partner’s website, the domain must first be registered on Bridgement’s backend. You will receive a unique reference as mentioned above. Please contact Bridgement and exchange the domain(s) for which the partner wishes to enable the embedded sign-up. E.g www.example.com

Example Usage

The following is an example of a button that will initiate the embedded sign-up process when a button is pressed.

<!DOCTYPE html>
<html>
    <head> </head>
    <body>
        <button onclick="bridgementSignUp()">Apply for Funding</button>
    </body>
    <script src="https://integration.bridgement.com/embedded-sign-up/v1/index.js"></script>
    <script>
        embeddedSignUp.settings = {
            clientReference: "supplied-company-reference"
        };
        
        function bridgementSignUp() {
            embeddedSignUp.signup();
        }
    </script>
</html>
